Project 1 is done for Achitecture of Computer Systems in Warsaw Management School.
The purpose of the project is to enable and disable LED in order to show in Morse Alphabet my name and surname.

Connection schema:
pin 13 - output to connection pannel to row 20
in row 20 LED anoda
in row 23 LED katoda
in row 23 rezystor 200
in row 27 rezystor 200
cable from row 27 to GND
